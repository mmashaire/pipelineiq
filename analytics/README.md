# Analytics

Analytics logic is implemented in `src/pipelineiq/funnel_metrics.py`.

The current baseline produces:

- funnel KPIs for the end-to-end pipeline
- campaign performance by sends, engagement, conversions, and revenue
- segment performance to compare audience value and progression
- recommendation outputs that turn metrics into next-best actions

This folder is also a good place for SQL assets and analysis notes in later phases.
